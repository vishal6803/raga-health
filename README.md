# Healthcare SaaS Dashboard | Raga.AI Assignment

This is a patient management dashboard built for the Raga.AI frontend assignment. It includes user authentication, data charts, and real-time notifications.

## 🔗 Live Link

**[raga-health-eight.vercel.app](https://raga-health-eight.vercel.app)**

---

## 🛠 Tech Stack

- **Core**: React 19 + TypeScript
- **State Management**: Zustand (Auth persistence & UI state)
- **Database & Auth**: Firebase
- **Styling**: Tailwind CSS v4 + shadcn/ui
- **Charts**: Recharts
- **Icons**: Lucide React
